Artifact 2 – AI Data Analysis Project

This project demonstrates how generative AI tools can assist with data analysis and identifying trends within datasets.

Project Overview

The analysis focused on identifying relationships between variables such as attendance, study hours, and academic performance. Using AI-assisted tools helped clean the dataset, generate visualizations, and detect correlations that may not be immediately obvious.

Skills Demonstrated

  • Data Analysis
  • AI Prompt Engineering
  • Pattern Identification
  • Data Visualization

Project Outcome

The project showed that increased study hours and higher attendance were positively correlated with improved academic performance. This demonstrates how AI tools can support faster and more efficient analysis of educational data.

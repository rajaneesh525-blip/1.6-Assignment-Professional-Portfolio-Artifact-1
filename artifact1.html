Artifact 1 – AI Fitness Chatbot

The AI Fitness Chatbot project was designed to explore how conversational AI can support users in maintaining healthy habits. The chatbot provides fitness tips, workout recommendations, and motivational responses based on user questions.

Project Goal

The goal of this project was to understand how AI-powered conversational systems can assist individuals in achieving personal fitness goals through interactive dialogue.

Technologies and Skills Used

  • Chatbot Design
  • User Interaction Flow
